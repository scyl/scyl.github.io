Pi Temerature Humidity Server

A server that logs data from a temperature and a humidity sensor on a Raspberry Pi and serves a webpage that shows the last 48 hours of data on a graph. The graph will automically update with new records.

This server uses a DS18B20 temperature sensor along with a DHT22 humidity sensor. Data from the sensors are logged at 5 minutes intervals via a Python script on to a SQLite database. A webpage that makes Ajax calls to the server and present the data in highcharts is also included. The server is written in Node.JS

Dodge Collector

Dodge Collector is a university assignment I made in a pair with the goal of creating a fast paced game that focuses on control, motion and continuously rising tension. We were required to have the user control an avatar and have objects in the game which the player has to dodge and also some objects that needs to be collected. We combined the two requirements and created Dodge Collector, a game where the player has to first dodge an incoming enemy who then turns into points that needed to be collected. See full documentation here.

I mainly worked on the idea and the coding/technical side of the game. My partner Kriska Budiman focused on designing the game's assets and finding suitable sound effects and music.
